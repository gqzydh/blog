前端性能优化

原则

  • 减少 CPU 计算， 减少网络请求
  • 多使用内存，缓存或者其他方法

加载资源优化

  1. 静态资源压缩合并；
  2. 静态资源缓存；
  3. 使用CDN让资源加载更快；
  4. 使用SSR 后端渲染，数据直接输出到HTML中；

渲染优化

  1. CSS 放前面，js放后面；
  2. 懒加载（lazyload, 图片懒加载，下拉加载更多）；
  3. 减少DOM查询，对DOM查询做缓存；
  4. 减少DOM操作，多个操作尽量合并在一起执行；
  5. 事件节流
  6. 尽早执行操作（如 DOMContentLoaded)；
