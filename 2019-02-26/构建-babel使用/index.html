Babel 是一个 JavaScript 编译器。
Babel 是一个工具链，主要用于将ECMAScript2015+代码转换为当前和旧版浏览器或环境中的向后兼容版本的JavaScript。

Babel主要做的事情

  • 转换js语法
  • 目标环境中缺少Polyfill功能\
  • 源代码转换
  • 更多

创建 .babelrc 配置文件

虽然已经配置好了 Babel ，但并没有让它真正生效。在项目的根目录中创建一个 .babelrc 文件并启用一些插件。

babel 升级到7.X

babel升级到7.X， 舍弃了以前的 babel-- 的命名方式，改成了@babel/-
