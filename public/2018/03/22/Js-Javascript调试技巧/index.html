console(控制台)基础用法

技巧 #1: console.trace()

你通过使用 console.trace() 获取堆栈跟踪和日志数据 ，从而知道日志的提示位置。

技巧 #2： console.time() 和 console.timeEnd()

如果你试图查找一个比较隐匿的性能问题，请使用 console.time() 开始计算时间，然后使用 console.timeEnd() 进行打印。
