  1. Bootstrap Maxlength

    允许输入用户字符数.它可以让你显示字符用户插入的最大长度。

  2. Bootbox.js

    一个小型的 JavaScript 库用来创建简单的可编程对话框，基于 Twitter 的 Bootstrap 开发。

  3. Bootstrap Modal

    模态框（Modal）是覆盖在父窗体上的子窗体。通常，目的是显示来自一个单独的源的内容，可以在不离开父窗体的情况下有一些互动。子窗体可提供信息、交互等。
