Lodash 常用API

2019_07_04 |
JS
| JS Lodash

原生用法

直接使用的API,基础技能

_.reject

根据条件去除某个元素。

// bar = [{id: 1, name: “bbb”, age: 25}]

_.pick

根据第二个参数的key的数组，筛选第一个参数中的值并返回

//bar = {name: “aaa”, age: 33}

_.keys

返回object中的所有key

//bar = [‘id’, ‘name’, ‘age’]

_.cloneDeep

深度拷贝，这个不用多说了吧，js中基础类型以外的类型，都会默认拷贝备份
var bar = _.cloneDeep(foo)

_.find

查找数组, 注意一下如果没找到的话，会返回undefined，要处理一下

//bar = {id: 0, name: “aaa”, age: 33}

_.keyBy

以某个属性为键，将数组转为对象

//bar = {
aaa: {id: 0, name: “aaa”, age: 33},
bbb: {id: 1, name: “bbb”, age: 25}}

_.filter

根据条件过滤出符合条件的元素，返回新数组

bar = {aaa: {id: 0, name: “aaa”, age: 33}

_.map

从集合中挑出一个key，将其值作为数组返回

//bar = [“aaa”, “bbb”]

.max/.min/_.sum

数组中最大值、最小值、数组求和

.pad/.padStart/_.padEnd

在两端、开头、末尾补齐字符

组合用法

再来几个炫酷的组合技能

选出json数组中id最大的一项

// bar = {id: 1, name: “bbb”, age: 25}

更新json数组中某一项的值

// bar = [
{id: 0, name: “ccc”, age: 33},
{id: 1, name: “bbb”, age: 25}]
