tsconfig.json 文件说明

  • compilerOptions ——编译器选项；
  • target ——编译目标，如这里编译为es5代码；
  • module ——处理独立文件时的模块加载方式；
  • sourceMap ——是否创建map文件以帮助调试；
  • exclude——编译器会编译TypeScript文件（.ts或.tsx），通过排除设置里的TypeScript文件不会被编译。
  • allowjs –为true,就可以ts和js混合用了.
  • outDir –将输出结构重定向到目录

extends和implements的区别

  • extends 后面跟的是类，表示继承关系，只能继承一个
  • implements 后面跟的是接口，表示实现接口，可以实现多个接口。实现接口就是实现接口中定义的方法,接口可以看作一个标准，比如定义了一个人的接口，那么人有吃饭，睡觉这个办法，你就可以用implements实现这个方法。
