自定义事件

Vue实例实现了一个自定义事件接口，用于在组件树中通信。这个事件系统独立于原生DOM事件，用法也不同。

  • 每个Vue实例都是一个事件触发器:
    • 使用 $on() 监听事件
    • 使用 $emit() 在它上面触发事件
    • 使用 $dispatch() 派发事件，事件沿着父链冒泡
    • 使用 $broadcast() 广播事件，事件向下传导给所有的后代
      不同与DMO事件，Vue事件在冒泡过程中第一次触发回调之后自动停止冒泡，除非回调明确返回true。
