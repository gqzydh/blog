vue2.0 组件库

2019_03_06 |
Vue.js
| Vue.js

将常用到的 Vue 组件汇总

UI组件

  • Element - 饿了么出品的Vue2的web UI组件库
  • Vux - 基于Vue.js 移动端 UI 组件库
  • Mint-UI - 基于 Vue.js 的移动端组件库
  • iviewui - 基于 Vue.js 的高质量 UI 组件库
  • Keen-UI - 轻量级的基本UI组件合集
  • vue-material - 通过Vue Material和Vue 2建立精美的app应用
  • muse-ui - 基于Vue 2.0三端样式一致的响应式 UI 库
  • vuetify - 为移动而生的Vue JS 2组件框架
  • vonic - 基于 vue.js 和 ionic 样式的 UI 框架，用于快速构建移动端单页应用
  • Radon-UI - 快速开发产品的Vue组件库

功能组件

  • eme - 优雅的 Markdown 编辑器。
  • vue-multiselect - Vue.js选择框解决方案
  • vue-table - 简化数据表格,编辑，分页
  • VueCircleMenu - 漂亮的vue圆环菜单
  • vue-waterfall - Vue.js的瀑布布局组件

Dome

  • vue-chat - vuejs和vuex及webpack的聊天示例
